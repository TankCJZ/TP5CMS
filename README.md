TP5CMS，一个基于ThinkPHP5.1版本开发内容管理系统,扁平化自适应后台系统
===============

### 主要功能：

 + 数据缓存
 + 权限控制
 + 自定义模型
 + 内容管理
 + 广告管理
 + 会员管理（未完成）
 + 站点设置
 + 日志记录
 + 数据库管理
 + 管理员分组

### 使用须知

> 运行环境要求PHP5.6+
